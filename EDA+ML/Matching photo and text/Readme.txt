Матчинг изображений с описанием

Введение

В данной работе исследуется задача матчинга изображений с их описанием. 

Особенность данной работы заключается в том, что:
- обработка нескольких датасетов,  и выбор типа их объединения на основе анализа содержащихся данных.
- использование предобученных трансформеров для обработки текста и изображений
- Использование классических моделей на задаче для нейронных сетей (работа с текстом и изображениями)

Для итоговой валидации была создана специальная Функция тестирования, которая принимала на вход текстовый запрос и возвращает изображение, которое наиболее подходит под него.

  Для машинного обучения в данном блокноте были использованы следующие модели:

1. KNeighborsRegressor (K ближайших соседей для регрессии)

2.  LinearRegression (Линейная регрессия)

3. Нейронная сеть

   - Библиотеки: `torch`, `torch.nn`
   - EfficientNet -
   - PyTorch Lightning (фреймворк для обучения нейронных сетей)

Дополнительно, были использованы различные инструменты и библиотеки для предобработки данных, работы с изображениями и текстом, а также для визуализации и оценки моделей.
